About The App

This is the website with collections of top readed books. First the data set from kaggle is taken to do analysis. Popularity based System and Collaborative based System is Used. With Popularity based system the top 10 books are retrieved on the basis of rating.

Similarly all data set is used to build the Collaborative Recommendation system, which recommends 5 books on the basis of the given book.

  • To see that, you need to click "More Details" on any particular book
